# 게임 룰 상세 (Game Rule) - 초안

`docs/PRD.md`에 정리된 기본 규칙을 기반으로 상세화한 문서다.

## 1. 플레이어

- 좌우 이동: 방향키 `←` / `→`
- 발사: `스페이스바` (수직 상방으로 작살 발사)
- 시작 시 무기는 싱글 샷(작살 1개) 상태이며, 화면에는 동시에 1발만 존재할 수 있다 (원작 기준).
- 플레이어가 풍선(모든 크기 포함)에 접촉하면 목숨을 1개 잃고, 스테이지가 재시작된다.
- 목숨이 모두 소진되면 게임 오버 처리된다.

## 2. 풍선(적) 규칙

- 풍선은 최대 4단계 크기를 가진다: 대형 → 중형 → 소형 → 극소형.
- 작살에 맞은 풍선은 한 단계 작은 풍선 2개로 분열된다 (대형→중형 2개, 중형→소형 2개, 소형→극소형 2개).
- 극소형 풍선이 작살에 맞으면 분열 없이 소멸한다.
- 풍선은 중력의 영향을 받아 포물선을 그리며 바닥/벽/천장에 부딪혀 튕긴다.
- 풍선 크기가 작을수록 이동 속도가 빨라진다 (원작 특징 참고).

## 3. 스테이지 클리어 / 진행

- 스테이지에 존재하는 모든 풍선(분열로 생성된 풍선 포함)을 제거하면 해당 스테이지를 클리어한다.
- 클리어 시 다음 스테이지(미션)로 진행한다.

## 4. 점수 (초안, 추후 구체화 필요)

- 풍선을 제거할 때마다 점수를 획득한다.
- 작은 풍선일수록 더 높은 점수를 획득하는 방식을 고려한다 (세부 수치는 추후 결정).

## 5. 이번 범위에서 제외되는 사항

- 파워업 아이템(더블 와이어, 벌컨 미사일, 시계, 모래시계 등)은 이번 초안에 포함하지 않는다.
- 스테이지별 배경/장애물 기믹은 미션 1 이후 확장 시 논의한다.
