# 메인 화면 (Main Screen) - 초안

## 1. 목적

게임 실행 시 최초로 보여지는 화면으로, 플레이어가 게임을 시작하거나 기본 정보를 확인할 수 있도록 한다.

## 2. 구성 요소

- **타이틀 로고**: 화면 상단 중앙에 "PANG" 게임 타이틀 표시
- **시작 버튼 / 안내 문구**: "스페이스바를 눌러 시작" 등 게임 시작 트리거 안내
- **조작 방법 안내**: 방향키(←/→) 이동, 스페이스바 발사 등 기본 조작법 간단 표기
- **배경**: 간단한 정적 배경 또는 원작 분위기를 참고한 배경 이미지/색상

## 3. 동작 흐름

1. 페이지 로드 시 메인 화면이 표시된다.
2. 플레이어가 시작 트리거(스페이스바 또는 시작 버튼)를 입력하면 미션 1(첫 스테이지) 화면으로 전환된다.

## 4. 이번 범위에서 제외되는 사항

- 로그인/회원가입, 랭킹, 사운드/옵션 설정 등 부가 메뉴는 이번 초안 범위에 포함하지 않는다.
- 상세 UI 디자인(색상, 폰트, 애니메이션 등)은 이후 구체화한다.
