# 메인 화면 (Main Screen) - 초안

## 1. 목적

게임 실행 시 최초로 보여지는 화면으로, 플레이어가 게임을 시작하거나 기본 정보를 확인할 수 있도록 한다.

## 2. 구성 요소

- **타이틀 로고**: 화면 상단 중앙에 "PANG" 게임 타이틀 표시
- **게임 선택 메뉴**: 미션 목록(현재는 "미션 1"만 존재)을 표시하며, 그중 하나가 항상 "선택된" 상태를 가진다.
- **시작 / 게임 종료 메뉴 항목**: 게임 선택 메뉴 아래에 위치한 실행 항목으로, 메뉴 탐색 대상에 포함된다.
- **조작 방법 안내**: 방향키/스페이스바 등 메뉴 조작법을 화면에 간단히 표기한다.
- **배경**: 간단한 정적 배경 또는 원작 분위기를 참고한 배경 이미지/색상

## 3. 메뉴 조작 방법 (키보드)

- `↑` / `↓`: 메뉴 항목(미션 목록, 시작, 게임 종료) 간 선택 이동. 목록 끝에서는 반대쪽 끝으로 순환하거나 멈추는 방식 중 택1 (구현 시 확정).
- `스페이스바` 또는 `Enter`: 현재 선택된 메뉴 항목 실행 (미션 선택 시 해당 미션이 선택 상태로 표시되고, "시작" 선택 시 게임 화면으로 전환, "게임 종료" 선택 시 종료 동작 수행).
- 마우스 클릭도 함께 지원한다 (키보드 조작을 대체하는 것이 아니라 병행 지원).

## 4. 동작 흐름

1. 페이지 로드 시 메인 화면이 표시되고, 첫 번째 메뉴 항목이 기본 선택 상태가 된다.
2. 플레이어가 방향키로 메뉴 간 이동, 스페이스바/Enter(또는 클릭)로 선택 항목을 실행한다.
3. "시작"이 선택/실행되면 현재 선택된 미션(현재는 미션 1) 화면으로 전환된다.
4. "게임 종료"가 선택/실행되면 종료 동작(웹 환경에 맞는 안내 또는 처리)이 수행된다.

## 5. 비주얼 디자인 방향

- 90년대 아케이드/콘솔 게임 감성을 참고한 레트로 스타일로 구성한다 (예: 픽셀 폰트 또는 도트풍 서체, 선명한 원색 계열의 배색, 두꺼운 테두리, 네온/글로우 느낌의 강조색).
- 복고풍이되 가독성과 완성도를 해치지 않는 "나름 이쁜" 수준을 목표로 하며, 과도한 장식(깜빡임 남발, 지나친 스캔라인 효과 등)은 지양한다.
- 현재 선택된 메뉴 항목은 색상 반전, 테두리 강조, 또는 좌우 화살표 커서(`▶ 항목`) 등 레트로 게임에서 흔히 쓰이는 방식으로 시각적으로 구분한다.

## 6. 이번 범위에서 제외되는 사항

- 로그인/회원가입, 랭킹, 사운드/옵션 설정 등 부가 메뉴는 이번 초안 범위에 포함하지 않는다.
- 실제 폰트/색상 값 등 세부 디자인 토큰 확정은 설계 문서(`docs/design/`)에서 구체화한다.
