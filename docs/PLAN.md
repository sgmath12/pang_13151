# PLAN (Phase별 목표) - 초안

각 Phase는 실행 가능한(동작하는) 산출물을 목표로 하며, Phase 종료 시점마다 사람이 직접 실행해보고 해당 기능을 테스트할 수 있는 상태여야 한다.

---

## Phase 1. 메인 화면 UI

#### 목표 기능
- `npm run dev`로 접속 시 메인 화면(`docs/FEATURES/main.md` 기준)이 표시된다.
- 타이틀, "시작"/"게임 종료" 메뉴 항목이 화면에 노출된다.
- 방향키(↑/↓)로 메뉴 항목("시작", "게임 종료") 간 선택 이동이 가능하고, 현재 선택된 항목이 시각적으로 구분되어 보인다. (이 단계에서는 스페이스바/Enter로 실제 실행되는 동작까지는 없어도 됨)
- 90년대 레트로 게임 감성을 참고한 비주얼(도트풍 폰트, 원색 배색, 선택 커서 표시 등)로 구성한다.

#### 고객님이 확인할 부분
- 첫 화면 진입 시 타이틀, 시작, 게임 종료 항목이 모두 잘 보이는지.
- 방향키(↑/↓)로 메뉴 간 이동이 되고, 선택된 항목이 눈에 띄게 구분되는지.
- 전체적인 디자인이 레트로 게임 느낌을 주면서도 깔끔하게 보이는지.

---

## Phase 2. 메인 화면 버튼 동작

#### 목표 기능
- 시작 버튼 클릭 시 게임 화면(빈 화면 또는 임시 화면)으로 전환된다.
- 게임 종료 버튼 클릭 시 종료 안내(또는 메인으로 복귀 등 웹 환경에 맞는 처리)가 동작한다.

#### 고객님이 확인할 부분
- 시작 버튼을 눌렀을 때 화면 전환이 정상적으로 일어나는지.
- 게임 종료 버튼을 눌렀을 때 의도한 동작(안내 문구/복귀 등)이 일어나는지.

---

## Phase 3. 플레이어 좌우 이동

#### 목표 기능
- 게임 화면에 플레이어 캐릭터가 표시된다.
- 방향키(←/→)로 좌우 이동이 동작한다.
- 화면 경계 밖으로 나가지 않도록 처리한다.

#### 고객님이 확인할 부분
- 방향키 이동이 자연스러운지 (속도, 화면 끝에서 멈추는지).

---

## Phase 4. 작살 발사

#### 목표 기능
- 스페이스바로 작살을 수직 상방으로 발사한다.
- 발사체가 화면 상단을 벗어나면 사라진다.

#### 고객님이 확인할 부분
- 스페이스바로 작살이 발사되는지, 연타 시 동작이 이상하지 않은지.

---

## Phase 5. 풍선 등장 및 물리 동작

#### 목표 기능
- 미션 1 기준(`docs/FEATURES/mission1.md`) 풍선 1~2개가 화면에 등장한다.
- 풍선이 중력의 영향을 받아 바닥/벽/천장에 부딪히며 포물선으로 튕긴다.
- 아직 충돌 판정(작살/플레이어)은 없는 단계.

#### 고객님이 확인할 부분
- 풍선이 화면에 정상적으로 표시되는지 (크기, 위치).
- 풍선의 움직임이 자연스럽게 튕기는지 (원작과 유사한 느낌인지).

---

## Phase 6. 작살-풍선 충돌 및 분열

#### 목표 기능
- 작살이 풍선에 맞으면 `docs/FEATURES/game_rule.md`의 분열 규칙(대형→중형→소형→극소형)대로 분열/소멸한다.

#### 고객님이 확인할 부분
- 풍선을 맞췄을 때 크기별로 정확히 분열되는지, 최소 크기에서는 사라지는지.

---

## Phase 7. 플레이어-풍선 충돌 및 게임 오버 처리

#### 목표 기능
- 플레이어가 풍선(모든 크기)에 닿으면 목숨이 1개 감소하고 스테이지가 재시작된다.
- 화면에 남은 목숨 수가 표시된다.
- 목숨이 0이 되면 게임 오버 화면이 표시되고, 게임 오버 화면에서 메인 화면으로 돌아갈 수 있다.

#### 고객님이 확인할 부분
- 풍선에 닿았을 때 목숨이 줄고 스테이지가 재시작되는지, 남은 목숨 표시가 정확히 갱신되는지.
- 목숨을 모두 소진했을 때 게임 오버 화면이 뜨고, 메인으로 정상적으로 돌아가는지.

---

## Phase 8. 미션 1 클리어 처리

#### 목표 기능
- 화면에 존재하는 모든 풍선(분열 포함)을 제거하면 미션 1 클리어로 판정된다.
- 클리어 안내 화면(또는 문구)이 표시된다.

#### 고객님이 확인할 부분
- 모든 풍선을 제거했을 때 클리어 판정이 정확한 타이밍에 뜨는지.

---

## Phase 9. 전체 플레이 흐름 통합

#### 목표 기능
- 메인 화면 → 미션 1 플레이 → 클리어/게임 오버 화면 → 메인 화면 복귀까지 전체 흐름이 끊김 없이 동작한다.
- 반복 플레이(재도전)가 정상적으로 동작한다.

#### 고객님이 확인할 부분
- 처음부터 끝까지 전체 흐름을 플레이했을 때 막히거나 어색한 부분이 없는지.
- 여러 번 반복 플레이해도 상태가 꼬이지 않고 안정적으로 동작하는지.

---
