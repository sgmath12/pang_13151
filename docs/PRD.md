# PANG 게임 PRD (Product Requirement Document)

## 1. 개요

**팡(Pang)**은 1989년 Mitchell Corporation이 개발한 아케이드 게임으로, 일본에서는 "퐁핑 월드(Pomping World)", 북미에서는 "Buster Bros."라는 이름으로도 알려져 있다. 화면 위아래로 튕겨 다니는 풍선(또는 공)을 작살(하푼)로 터뜨려 스테이지를 클리어하는 액션 게임이다.

이 프로젝트에서는 팡의 핵심 규칙을 참고하여 웹 기반으로 재구현하며, 이번 단계에서는 **미션 1(스테이지 1)** 수준의 기본 플레이 경험을 목표로 한다.

## 2. 게임 목표

- 플레이어는 화면 하단에서 좌우로 이동하며 위로 작살을 발사한다.
- 스테이지 내 모든 풍선을 제거하면 클리어, 다음 스테이지로 진행한다.
- 풍선(또는 그 파편)에 닿으면 플레이어는 목숨을 잃는다.

## 3. 기본 규칙 (미션 1 기준)

- **풍선 분열**: 큰 풍선이 작살에 맞으면 더 작은 풍선 2개로 분열되며, 이 과정이 최대 3단계까지 반복된다. 가장 작은 단계의 풍선이 터지면 완전히 사라진다.
- **풍선 물리**: 풍선은 바닥과 벽에 부딪히며 포물선을 그리듯 튕겨 다닌다.
- **플레이어 조작**: 방향키(←/→)로 좌우 이동, 스페이스바로 수직 방향 작살 발사를 하며, 시작 시 작살은 1개(싱글 샷)이다.
- **목숨/재도전**: 플레이어가 풍선에 닿으면 목숨을 잃고, 목숨을 모두 소진하면 게임 오버된다.
- **클리어 조건**: 스테이지에 남아있는 풍선(모든 분열 단계 포함)을 전부 제거하면 스테이지 클리어.

## 4. 미션 1(스테이지 1)의 특징

- 초기 스테이지답게 등장하는 풍선의 개수와 크기가 적고 단순하여, 기본 조작(이동, 발사, 풍선 회피)을 익히는 튜토리얼 성격을 가진다.
- 큰 풍선 1~2개로 시작해 분열을 거치며 점차 풍선 개수가 늘어나는 구조로, 분열 매커니즘을 자연스럽게 체험하게 한다.
- 별도의 장애물이나 특수 기믹 없이 "풍선 제거"라는 핵심 목표에만 집중된 구성이다.

## 5. 향후 확장 여지 (참고용, 이번 범위 아님)

원작에는 풍선을 터뜨릴 때 드롭되는 파워업(더블 와이어, 파워 와이어, 벌컨 미사일, 시간 정지 시계, 모래시계 등)과 스테이지별 배경/장애물 요소가 존재한다. 이는 이후 미션 확장 시 참고 대상으로 남겨둔다.
